Thermal Printer
Remote Printer Backend

Languages/Technologies: PHP, SQL, XML

I created a backend web server for a Doordash like service to interact with the Epson tm-t88vi thermal printer API to delegate print jobs to multiple remote restaurants using polling to avoid network conflicts. This enabled printers to be set up without requiring port forwarding or advance networking. My backend would take print jobs and format them into the expected XML format required of the API. After it would take poll request from each printer, match the print job to their ID and send off the print job

Online Stock
Monitor

Languages/Technologies: PYTHON, SQL

I created a multithreaded bot that would monitor the stock of popular websites and post stock updates through webhooks using JSON. You could subscribe to stock updates by inputting a SKU for each item which then it would scrape the API of each site and post a notification on any change.
